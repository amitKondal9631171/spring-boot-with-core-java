1.) What is a Lambda Expression, and why use them?
	It’s a function that can be referenced and shared as an object. Lambda Expressions require less coding.

What is a functional interface?
    A functional interface is an interface that contains just one abstract method.

How are functional interfaces and Lambda Expressions related ?
	Lambda expressions are applied only to the functional interface’s abstract method.

What is a stream, and how does it differ from a collection?
	A stream is an iterator whose function is to accept a set of actions and apply them to each of the elements it contains.
